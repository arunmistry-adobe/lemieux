# Promo Cards

Displays a row of 1–3 full-bleed promotional image cards with overlay text and a CTA button. Used as the homepage hero section.

## Content Structure

Each row is one card:

| Image | Content |
|-------|---------|
| Picture | Heading (h2–h4), subtext (p), CTA link (a) |

## Example

| Promo Cards | |
|---|---|
| [image] | **Outfit Builder** / Create your style / [Build Your Outfit](/outfit-builder) |
| [image] | **Shine In** / Brilliance Collection / [Shop Now](/collections/brilliance) |
| [image] | **Download Now** / LeMieux Insiders App / [Get the App](/app) |

## Behavior

- On mobile, cards stack vertically (1 column).
- On tablet and above (≥600px), cards display in a 3-column grid.
- The entire card is clickable — clicking anywhere navigates to the CTA link href.
- Images have a subtle zoom-in on hover.

## Styling

Text overlays use a bottom-to-top gradient. CTA buttons are white with dark text, turning sage green on hover.
